=== Pie Image Extractor ===
Requires at least: 5.8.0
Tested up to: 5.8.1
Requires PHP: 7.1.0


== Description ==

This plugin extracts images from posts and copies them to upload folder by post type and post name


== Changelog ==

See changelog.md


== Installation ==

Quick and easy installation:

1. Upload the folder 'pie-image-extrator' to your plugin directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You'll find the "Pie Image Extractor" in the admin menu
